# 🐍 贪吃蛇大作战 (Snake Game)

这是一个基于 HTML5 Canvas 和原生 JavaScript 开发的现代版贪吃蛇游戏。项目采用了时尚的暗色玻璃拟态风格（Glassmorphism），并结合 SVG 技术实现了生动的卡通视觉效果。

## ✨ 主要特性

### 🎨 视觉体验
*   **卡通化角色**：使用 SVG 绘制的可爱蛇头，自带灵动的大眼睛和腮红，且眼睛会跟随移动方向转动。
*   **精美道具**：将传统的圆点食物替换为精致的红苹果图标。
*   **现代 UI**：界面采用深空蓝背景搭配磨砂玻璃效果，视觉体验高端且舒适。
*   **动态特效**：包含平滑的渐变蛇身、发光效果以及动态背景球体。

### 🎮 游戏玩法
*   **经典核心**：通过键盘控制蛇移动吃苹果，每吃一个苹果身体变长，分数增加。
*   **难度进阶**：随着分数增加，游戏速度会自动提升，挑战性逐渐升级。
*   **完整流程**：支持游戏开始、暂停、接触墙壁或自身死亡判定、游戏结算。

### 👤 用户系统
*   **本地认证**：模拟了完整的注册与登录流程（数据存储在 LocalStorage）。
*   **数据记录**：自动记录用户的最高分和每一局的详细战绩（得分、时长）。
*   **历史查询**：提供专门的历史记录页面查看过往表现。

## 🛠️ 技术栈

*   **核心语言**: HTML5, CSS3, JavaScript (ES6+)
*   **渲染引擎**: HTML5 Canvas API
*   **图形素材**: SVG (Scalable Vector Graphics)
*   **数据存储**: Browser LocalStorage
*   **无后端**: 纯前端实现，无需服务器支持即可完整运行。

## 🚀 快速开始

本项目为纯静态网页应用，开箱即用。

1.  **获取代码**: 下载或克隆本项目到本地。
2.  **启动游戏**: 双击打开根目录下的 \`index.html\` 文件（建议使用 Chrome、Edge 或 Firefox 等现代浏览器）。
3.  **注册登录**: 首次进入请点击“注册”创建一个本地账号，登录后即可开始游戏。

## 🎮 操作指南

| 按键 | 功能 |
| --- | --- |
| **↑ / W** | 向上移动 |
| **↓ / S** | 向下移动 |
| **← / A** | 向左移动 |
| **→ / D** | 向右移动 |
| **空格键** | 暂停 / 继续 |

## 📝 开发备注

*   所有游戏数据均存储在浏览器的 LocalStorage 中，清除浏览器缓存会导致数据丢失。
*   项目使用了 ES6+ 语法，请确保浏览器版本不要过低。
